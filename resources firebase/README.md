# Resources

Ce dossier contient les ressources partagées pour le projet RouteWatch.

## Structure

- `firebase/`: Configuration et clés pour Firebase
  - `firebase-service-account.json`: Clé de compte de service Firebase (à remplacer par votre propre clé)
  - `README.md`: Instructions pour configurer Firebase

## Configuration Firebase

Pour configurer Firebase pour le projet, suivez les instructions dans le fichier `resources/firebase/README.md`.

⚠️ IMPORTANT: Ne commitez jamais le fichier `firebase-service-account.json` dans un dépôt public. Ce fichier contient des informations sensibles qui permettent d'accéder à votre projet Firebase.

## Documentation supplémentaire

Pour plus d'informations sur l'installation et la configuration de Firebase, consultez:

- `documentation/FIREBASE_INSTALLATION.md`: Guide d'installation de Firebase
- `documentation/FIREBASE_SYNC_GUIDE.md`: Guide de synchronisation Firebase
